### OFFER 1 — "입실 전 화이트글러브 청소 보증"

**문제: 청소 불량** (긴급 10 / WTP 9)

**🎯 이상적인 고객**

| 항목 | 정의 |
|------|------|
| 인구통계 | 20~30대 커플, 여행 목적 1~2박 로맨틱 숙박, 특별 기념일(생일·기념일·프러포즈) |
| 심리 | "설레는 마음으로 갔다가 청소 안 된 방에서 기분 잡치면 그 여행 전체가 망하는 거잖아요" |
| 핵심 욕구 | 첫인상에서 '완벽한 환경'을 경험해 여행 전체 감정 기조를 높이고 싶다 |

**💎 가치 제안**

**"체크인 2시간 전, 제3자 청소 검수 완료 인증서를 카카오톡으로 먼저 받으세요"**

단순히 청소가 된 숙소가 아니라, 체크인 전 3단계 독립 검수(청소팀 → 코디네이터 → 게스트 확인)를 거친 숙소입니다. 입실 즉시 문제가 있으면 30분 내 재청소 또는 전액 환불. 기대와 현실의 갭을 없앱니다.

**💰 가격 전략**

| 티어 | 내용 | 가격 |
|------|------|------|
| 기본 | 검수 완료 인증 + 입실 안내 키트 | 숙박비에 포함 |
| 스탠다드 | 기본 + 어메니티 업그레이드(바스솔트, 미니샴페인 1병) | +20,000원 |
| 프리미엄 | 스탠다드 + 전문 플로리스트 꽃 배치 | +50,000원 |

**🛡️ 보상 + 리스크 제거**

- 입실 후 청소 문제 발견 → 30분 내 재청소 팀 파견
- 재청소 불가 상황 → 다음 예약 숙박비 30% 크레딧 지급
- 호실 이동 강요 → 업그레이드 호실 제공 (동급 이상) 또는 전액 환불

**🏆 경쟁사보다 나은 이유**

| | 에어비앤비 일반 | 호텔 | 이 오퍼 |
|--|--|--|--|
| 청소 검증 | 호스트 자체 청소 | 하우스키핑팀 | 제3자 독립 검수 + 인증서 |
| 문제 발생 시 | 호스트 재량 | 프런트 민원 처리 | 30분 내 현장 해결 + 자동 보상 |
| 신뢰 증거 | 이전 게스트 후기 의존 | 브랜드 신뢰 | 체크인 전 실시간 상태 공유 |
| 감정 가치 | 불확실성 | 평균적 만족 | 설레는 첫인상 보장 |

### OFFER 2 — "침구 걱정 제로 패키지"

**문제: 침구 교체 비용 별도 청구 (2만원)** (긴급 7 / WTP 9)

**🎯 이상적인 고객**

| 항목 | 정의 |
|------|------|
| 인구통계 | 25~35세 커플, 주말 1박 여행, 예산 의식적이지만 '깔끔함'은 비필수 타협 불가 항목 |
| 심리 | "2만원이 문제가 아니라 왜 당연한 걸 돈 받냐는 게 기분 나쁜 거죠. 호텔이면 당연히 교체해주는데" |
| 핵심 욕구 | 숨겨진 추가 비용 없는 투명한 거래, '대우받는 느낌' |

**💎 가치 제안**

**"침구 교체? 당연히 무료입니다. 호텔처럼요."**

숙박 요금에 전체 침구 세트 교체가 포함됩니다. 추가 요금 없음. 체크인 시 봉인된 새 시트·베개커버·이불커버 세트를 직접 확인하세요. 위생 강박이 있는 분을 위해 의료용 비닐 포장 침구 옵션도 제공합니다.

**💰 가격 전략**

| 티어 | 내용 | 가격 |
|------|------|------|
| 스탠다드 | 기본 침구 세트 교체 포함 | 숙박비에 포함 |
| 프리미엄 | 호텔 등급 이집션 코튼 1000수 침구 세트 | +15,000원 |
| 럭셔리 | 프리미엄 + 침대 구성 서비스(웰컴 초콜릿, 턴다운 서비스 연출) | +35,000원 |

**🛡️ 보상 + 리스크 제거**

- 침구 위생 불만족 → 즉시 교체 또는 다음 예약 10% 할인 쿠폰
- 추가 비용 청구 발생 시 → 해당 금액 × 3 환불 보장

**🏆 경쟁사보다 나은 이유**

| | 에어비앤비 일반 | 호텔 | 이 오퍼 |
|--|--|--|--|
| 침구 교체 비용 | 별도 2만원 청구 (업계 관행) | 무료 포함 | 무료 + 봉인 위생 포장 |
| 투명성 | 체크인 후에야 정책 인지 | 예약 전 명시 | 예약 페이지에서 명시 + 사진 증거 |
| 위생 신뢰 | 이전 게스트 사용 여부 불명 | 하우스키핑 처리 | 봉인 포장으로 미개봉 확인 가능 |

### OFFER 3 — "완전 프라이버시 욕실 보증 스테이"

**문제: 욕실·샤워 공간 블라인드/프라이버시 미비** (긴급 8 / WTP 8)

**🎯 이상적인 고객**

| 항목 | 정의 |
|------|------|
| 인구통계 | 20~30대 여성이 포함된 커플, 오션뷰 선호, 야간 조명 속 욕실 사용 중요시 |
| 심리 | "오션뷰 좋아서 예약했는데 샤워할 때 밖에서 보일 것 같아서 불안해서 제대로 못 씻었어요. 로맨틱이 다 날아가버렸죠" |
| 핵심 욕구 | 외부 시선 차단 + 오션뷰를 동시에 즐기는 안전한 프라이빗 공간 |

**💎 가치 제안**

**"바다를 보며 샤워해도 아무도 못 봅니다 — 편광 필름 + 블라인드 이중 차단"**

욕실 창문에 주간 단방향 편광 필름(안에서는 바다 보임, 밖에서는 불투명) + 야간 전동 블라인드를 적용했습니다. 오션뷰를 포기하지 않고 완전한 프라이버시를 누릴 수 있습니다. 입실 시 블라인드 조작법 안내 영상을 QR코드로 제공합니다.

**💰 가격 전략**

| 티어 | 내용 | 가격 |
|------|------|------|
| 기본 | 편광 필름 + 수동 블라인드 적용 객실 | 숙박비에 포함 |
| 스파 업그레이드 | 기본 + 욕실 입욕제 세트 + 조명 디밍 리모컨 | +25,000원 |
| 로맨틱 패키지 | 스파 업그레이드 + 거품 목욕 셋업 + 캔들 | +55,000원 |

**🛡️ 보상 + 리스크 제거**

- 프라이버시 불만족(외부 시선 우려) → 당일 추가 암막 커튼 설치 무료
- 블라인드 작동 불량 → 30분 내 수리 또는 다음 예약 15% 환불

**🏆 경쟁사보다 나은 이유**

| | 에어비앤비 일반 | 호텔 | 이 오퍼 |
|--|--|--|--|
| 욕실 프라이버시 | 호스트 재량, 검증 불가 | 표준 욕실 창문 없음/불투명 | 오션뷰 유지 + 이중 차단 보증 |
| 야간 안심 | 불안 상태로 사용 | 프라이버시 확보되나 뷰 없음 | 뷰 + 프라이버시 동시 충족 |
| 로맨틱 경험 | 불안이 감정 방해 | 무난하지만 평범 | 목욕 자체가 여행의 하이라이트 |

### OFFER 4 — "사계절 온기 보장 스테이 (겨울 특화)"

**문제: 단열 불량 — 창문·침대 주변 한기 심함** (긴급 8 / WTP 8)

**🎯 이상적인 고객**

| 항목 | 정의 |
|------|------|
| 인구통계 | 11월~3월 여행 커플, 바다 근처 숙박 경험자, 오션뷰 숙소의 추위를 한 번 이상 경험한 적 있음 |
| 심리 | "바다 가까울수록 춥다는 거 알지만, 이불 뒤집어쓰고 버티는 건 여행이 아니잖아요. 그냥 집에 있는 거지" |
| 핵심 욕구 | 한겨울에도 따뜻한 포근함 속에서 오션뷰를 즐기고 싶다 |

**💎 가치 제안**

**"영종도 바닷바람이 와도 실내는 25°C — 3중 단열 보증 겨울 스테이"**

창문 단열 필름 + 침대 배치 최적화 + 온수매트 기본 제공으로 구성된 3중 겨울 솔루션. 체크인 시 객실 온도 25°C 세팅 완료 상태를 보장합니다. 입실 후 30분 내 18°C 이하라면 전기장판·추가 이불 즉시 제공 + 다음 예약 할인.

**💰 가격 전략**

| 티어 | 내용 | 가격 |
|------|------|------|
| 겨울 기본 | 단열 필름 + 온수매트 제공 (11~3월) | 숙박비에 포함 |
| 코지 업그레이드 | 기본 + 핫초코·유자차 웰컴 드링크 + 극세사 담요 추가 | +18,000원 |
| 허니문 윈터 | 코지 업그레이드 + 인바디 핫팩 세트 + 욕조 온도 유지 버블 입욕제 | +40,000원 |

**🛡️ 보상 + 리스크 제거**

- 입실 후 30분 내 설정 온도 미달 → 즉시 추가 난방 기기 제공
- 단열 체감 불만족 → 체크아웃 후 숙박비의 20% 환불
- 시설 고장(난방기 불량) → 대체 전기장판 즉시 제공 또는 전액 환불

**🏆 경쟁사보다 나은 이유**

| | 에어비앤비 일반 | 호텔 | 이 오퍼 |
|--|--|--|--|
| 겨울 단열 | 호스트 시설 운에 의존 | 중앙공조 표준 | 3중 단열 + 입실 전 25°C 세팅 |
| 추위 대응 | 게스트가 직접 요청해야 함 | 객실 조정 가능 | 자동 감지 + 30분 보증 |
| 감성 경험 | 한기 = 불쾌 기억 | 평범한 따뜻함 | 따뜻함이 로맨틱 무드를 완성 |

### OFFER 5 — "무결점 시설 + 레이트 체크아웃 패키지"

**문제: 시설 고장 방치 + 체크아웃 시간 압박** (긴급 8 / WTP 7, 긴급 6 / WTP 7)

**🎯 이상적인 고객**

| 항목 | 정의 |
|------|------|
| 인구통계 | 30~40대 커플, 1박 2일 여행, 아침 느긋함을 중요시하는 '슬로우 트래블' 선호층 |
| 심리 | "왜 11시에 쫓겨나야 해요? 아침 먹고 느긋하게 바다 한 번 더 보고 싶은데 그게 그렇게 큰 요구인가요?" |
| 핵심 욕구 | 아침 시간의 여유 + 모든 시설이 정상 작동하는 스트레스 없는 체류 |

**💎 가치 제안**

**"오전 13시 레이트 체크아웃 무료 + 시설 불량 제로 보증 — 마지막 아침까지 여행입니다"**

체크아웃 시간을 11:00에서 13:00으로 무료 연장. 모든 시설(도어락, 커튼, 샤워기, 난방기)은 분기별 정기 점검 완료 인증을 받은 객실만 운영합니다. 시설 고장 발생 시 30분 내 수리 또는 즉시 체크아웃 후 전액 환불.

**💰 가격 전략**

| 티어 | 내용 | 가격 |
|------|------|------|
| 기본 | 13시 레이트 체크아웃 포함 | 숙박비에 포함 |
| 브런치 업그레이드 | 기본 + 체크아웃 전 브런치 바구니 (크루아상·과일·드립커피) 객실 배달 | +30,000원 |
| 풀 데이 릴렉스 | 브런치 업그레이드 + 15시 체크아웃 + 수영장/루프탑 이용권 | +60,000원 |

**🛡️ 보상 + 리스크 제거**

- 시설 고장 발생 → 30분 내 수리 또는 숙박비 30% 환불
- 레이트 체크아웃 불가 통보(선예약 충돌) → 다음 예약 레이트 체크아웃 무료 + 커피 쿠폰
- 도어락·안전장치 오작동 → 즉시 호스트 출동 + 전액 환불

**🏆 경쟁사보다 나은 이유**

| | 에어비앤비 일반 | 호텔 | 이 오퍼 |
|--|--|--|--|
| 체크아웃 시간 | 11:00 표준 | 12:00 표준 | 13:00 무료 (업계 최고 수준) |
| 시설 관리 | 고장 후 대응 | 유지보수팀 상시 | 분기 사전 점검 인증 + 30분 SLA |
| 아침 경험 | 짐 싸느라 바쁨 | 빠른 퇴실 압박 | 마지막 아침도 여행의 일부 |
| 안전 | 잠금 불량 리스크 있음 | 카드키 표준 | 정기 점검 완료 객실만 운영 |

**훅 O-03**

**훅 O-04**

**훅 C-02**

**훅 C-03**

**훅 S-03**

## PART 3. 감정 트리거 맵

| 감정 | 트리거 메시지 | 반응 행동 |

| 공포 | "이전 투숙객의 흔적이 남은 침대", "화장실 갇힘 사고" | 지금 당장 안전한 숙소 검색 |
| 분노 | "침구 교체 2만원 별도 청구", "11시 퇴실 압박 문자" | 공유·댓글·저장으로 '나도 그랬어' 동조 |

| 공감 | "기념일 여행 망친 경험", "추위에 잠 못 잔 밤" | 댓글로 자기 경험 털어놓기, 저장 |

## PART 4. SPICE 구조 예시 3개

### SPICE 예시 1 — 청소 불량 (공포 감정)

- **Stop**: "영종도 커플 숙소 예약 전 이것만 확인하세요"
- **Pain**: "체크인 당일 청소 안 된 방, 다른 호실로 이동 강요 — 실제 후기에서 7번 반복된 경험입니다"
- **Insight**: "후기 별점은 청소 문제를 숨깁니다. 4.29점 숙소의 ★1 리뷰는 전부 청소+응대 문제였습니다"
- **Claim**: "저희는 체크인 2시간 전 제3자 검수 완료 인증서를 카카오톡으로 먼저 드립니다"
- **Engage**: "아래 링크에서 검수 체크리스트 항목 직접 확인하세요 (총 27개 항목)"

---

### SPICE 예시 2 — 침구 비용 (분노 감정)

- **Stop**: "침구 교체에 2만원 받는 에어비앤비, 정상인가요?"
- **Pain**: "고평점 숙소(4.88, 4.95점)에서도 불만 키워드로 반복 등장한 유일한 정책 — 침구 교체 비용"
- **Insight**: "게스트가 화난 건 2만원이 아닙니다. '당연한 걸 왜 따로 받냐'는 대우의 문제입니다"
- **Claim**: "저희 숙소는 봉인 포장 침구 세트를 무료로 제공합니다 — 추가 요금 없이, 호텔처럼"
- **Engage**: "예약 페이지에서 '침구 교체 무료 포함' 항목 확인 후 예약하세요"

---

### SPICE 예시 3 — 레이트 체크아웃 (공감 감정)

- **Stop**: "마지막 아침도 여행이어야 합니다"
- **Pain**: "11시 체크아웃 알림 문자, 아직 자고 있는데 짐 싸느라 보낸 마지막 아침 — 경험 있으신 분 많죠"
- **Insight**: "레이트 체크아웃을 무료로 제공한 숙소의 후기에서 '다시 오고 싶다' 언급률이 3배 높았습니다"
- **Claim**: "저희는 13시 레이트 체크아웃을 기본 제공합니다 — 마지막 아침 바다를 느긋하게 즐기세요"
- **Engage**: "다음 주말 예약 가능 날짜 확인해보세요 (링크)"
