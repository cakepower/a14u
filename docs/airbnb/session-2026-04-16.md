## Q3. 시장 분석

> 참조 파일: `docs/analysis-market-2026-04-16.md`

### TAM / SAM / SOM

| 구분 | 시장 정의 | 규모 추정 |
|------|----------|----------|
| TAM | 국내 전체 숙박 시장 (커플 여행 포함) | 약 20조 원/년 |
| SAM | 인천/영종도 권역 에어비앤비·단기 렌탈 커플 숙박 시장 | 약 600억~900억 원/년 |
| SOM | 커플 전용 큐레이션 채널로 현실적으로 공략 가능한 규모 | 약 30억~60억 원/년 |

### 핵심 트렌드 5개

1. **오션뷰 감성 > 숙소 스펙** — 전망 키워드 압도적 1위 (숙소1: 117회, 숙소3: 95회)
2. **호캉스의 일본감성·한옥감성 혼합** — 테마 정체성이 예약 전환율과 재방문율 향상
3. **늦은 체크아웃 = 재방문 결정 변수** — 무료 레이트 체크아웃은 최고 ROI 서비스 요소
4. **자쿠지·스파 필수 요소화** — 자쿠지 키워드 27회, 리뷰 6개 중 4개가 자쿠지 직접 언급
5. **슈퍼호스트 + 즉각 응답 = 예약 신뢰 프리미엄** — 재방문 의향 슈퍼호스트 숙소에 집중

### 덜 공략된 기회 5개

1. 커플 전용 큐레이션 예약 채널 — 에어비앤비 검색 파편화의 공백
2. 기념일 패키지 번들링 — 호스트 개별 운영 한계로 공급 부재
3. 영종도 노을 타임 콘텐츠화 — 숙소 외부 관광 정보 완전 공백
4. 저평점 숙소 리브랜딩 컨설팅 — B2B 호스트 운영 노하우 부족 시장
5. 반려동물 동반 커플 숙소 세그먼트 — 공급 절대적 부족, 연 20% 성장
