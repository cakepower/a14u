# 영종도 Airbnb 운영을 위한 지역 이벤트 정보 수집 플랜

> 목적: 숙소 홍보·가격 전략·게스트 안내에 활용할 영종도/인천 지역 이벤트 정보 수집 체계 구축  
> 작성일: 2026-04-12

---

## 1. 정보 수집 카테고리

영종도 Airbnb에 영향을 주는 이벤트는 크게 5가지로 나뉩니다.

| 카테고리 | 설명 | 수요 영향 |
|----------|------|-----------|
| 지역 축제·행사 | 을왕리 해수욕장, 영종도 레일바이크 이벤트 등 | 직접 수요 |
| 공항 연계 이벤트 | 인천공항 전시·팝업, 항공 취항 이벤트 | 경유 수요 |
| 인스파이어 리조트 | 콘서트, 스포츠, 공연 등 대형 이벤트 | 강한 직접 수요 |
| 인천 시내 광역 행사 | 인천펜타포트 록페스티벌, 송도 이벤트 등 | 간접 수요 |
| 계절·자연 이벤트 | 해수욕 시즌, 낙조 명소, 갯벌 체험 시즌 | 계절 수요 |

## 3. 수집 스케줄

```
매일 (자동)
├── 인스파이어 공식 사이트 티켓 신규 등록 체크
└── 인스타그램 #영종도 해시태그 신규 포스트 모니터링

매주 월요일
├── 인천 중구청 공지사항 확인
└── 대한민국 구석구석 이번 달 행사 업데이트 확인

매월 1일
├── 인천관광공사 월별 행사 캘린더 전체 수집
├── 공공데이터 API 다음 달 행사 데이터 다운로드
└── 인스파이어 다음 2개월 이벤트 확인 및 가격 전략 수립

분기 1회
└── 연간 주요 축제 일정 정리 (인천펜타포트, 을왕리 해수욕장 개장/폐장 등)
```

## 4. 영종도 연간 주요 이벤트 캘린더 (기준 정보)

| 시기 | 이벤트 | 수요 강도 |
|------|--------|-----------|
| 3~4월 | 인천대공원 벚꽃, 영종도 봄 나들이 시즌 | ★★★ |
| 5~6월 | 인천상륙작전 기념행사(9월), 황사 이후 나들이 | ★★ |
| 7~8월 | **을왕리·왕산 해수욕장 개장** (최성수기) | ★★★★★ |
| 9월 | 인천 상륙작전 기념일, 인천펜타포트 락페스티벌 | ★★★★ |
| 10월 | 인천 국제공항 개항 기념 행사, 단풍 시즌 | ★★★ |
| 11~12월 | 인스파이어 연말 공연·카운트다운 이벤트 | ★★★★ |
| 연중 | 인스파이어 K-POP 콘서트, 격투기 이벤트 등 | ★★★★ |

## 5. 수집 정보 활용 방법

```
이벤트 발견
    ↓
수요 예측 (규모, 대상, 이동 반경)
    ↓
가격 전략
 ├── 대형 이벤트 (인스파이어 콘서트 등): 2~3주 전부터 가격 인상, 최소 박수 설정
 ├── 중형 이벤트: 1~2주 전 소폭 인상
 └── 계절 행사: 시즌 전체 기간 가격 조정
    ↓
게스트 안내 가이드 업데이트
 └── "이 기간에 영종도에서 즐길 수 있는 것들" 안내 문서에 반영
```
